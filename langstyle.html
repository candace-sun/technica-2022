What is a programming language?


A programming language is a language used to write instructions for the computer. These languages include, but are not limited to, HTML, CSS, Java Script, Python, Java, and C++


Why is it needed?


Programming languages helps us create websites, and games and many more, much easier. People also create art with programming such as animations and 2D/3D drawings.
